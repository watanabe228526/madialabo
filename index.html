「Webコンテンツ作成」の成果

本テーマについて

このテーマではHTMLでWebページの構造を作り, CSSで見た目の装飾を作ってJavaScriptで動きのあるページを実現する方法を学びます。これらを組み合わせて Web ページを構成する方法を学びます。

プロジェクト課題について

プロジェクト課題では世界の天気のWebページを作成しました。誰にでもわかりやすいような天気情報表示ページを作成したいと思い、地図を表示させてチェックボックスをクリックするとその都市の天気を表示するページを作りました。.map-containerで地図画像とラベルを含む枠を作り、.city-checkboxで地図上の都市ラベルを作り、top, leftで場所を決めました。世界の都市の場所を検索し、一つ一つtop,leftのパーセンテージを調整しました。

数当てゲーム

課題3-1の数当てゲームでは、1〜10の乱数を答えとして設定し、ユーザが予想して当てるゲームを実装しました。変数yosoを4で固定してhantei関数内で予想回数や判定結果をコンソールに出力する処理を実装しました。課題4-1の数当てゲームでは、ユーザの入力値を取得して答えと比較し、回数に応じて結果メッセージを表示する処理を実装しました。また課題3-1とは違いコンソールだけではなくテキストとして表示させました。ゲーム終了判定も行いました。また、isGameOverという変数を作り、ゲームが終わったかどうか判断させました。それにより、その後ユーザーが正解した後に入力しても回数が飛ばないように工夫しました。

各授業回の学習事項

第1回WWWとHTML

第1回の授業では、ウェブページの構造とコンテンツを作成する方法について学びました。タグ、属性、および要素の使用方法を理解し、段落、見出し、リンクなどの基本的な構成要素を組み合わせてページを構築しました。また、フォームや画像の埋め込みなど、さまざまな要素を使用してインタラクティブなコンテンツを作成する方法も学びました。

演習1-4: HTMLのさまざまな要素

第2回CSSによるWebページの装飾

第2回の授業では、ウェブページを装飾する方法について学びました。HTML が最初に与えられて，次にその装飾をする CSS を記述するという流れです。ページに合わせた色、フォント、レイアウトを組み合わせてページを構築しました。

演習2-3:ウェブブラウザ説明ページの装飾

第3回JavaScriptの基本

第3回のJavaScriptの授業ではWebページに動きを加える言語で、ページ表示時や操作時に実行され、自動更新や動的な演出を実現します。またウェブ開発ツールを使ってデバックする方法なども学びました。

演習3-6:オブジェクトの参照

第4回DOMとJavaScript

第4回の授業では、ページに設置されているHTML要素の検索方法とテキスト変更方法について学びました。querySelectorを使って要素を取得し、textContent を用いて内容を書き換える方法を実践しました。これにより、動的にページの表示を更新する仕組みを理解しました。

演習4-10:足し算ページ

第5回イベント処理

第5回の授業では、Webコンテンツを表現するデータ構造ついて学びました。その中でも要素の編集などを学びました。Webページがどのように作られているかを学び、情報をわかりやすく整理して見せるための仕組みを理解することができました。

演習5-7:DOMプログラミング(2)

工夫点，苦心点，強調点，感想

この講義を通して、Webページの構造を記述するためのHTML、ページを装飾するCSS、複雑な機能を実現するJavaScript、Webコンテンツを構成するDOMについて理解を深めることができました。プロジェクト課題では、世界の天気情報を表示するWebページを作成しました。工夫点としては、世界地図の画像上にチェックボックスを配置し、それをクリックすることで都市ごとの天気情報が表示されるようにした点です。これは講義で学習したHTMLとCSSとJavaScript、DOMの知識を応用したものです。具体的には、内部スタイルシートのstyleタグを使い、.map-container クラスで地図と都市ラベルを含む枠を作成し、.city-checkbox クラスでは位置を絶対指定であるposition: absoluteを使い、topとleft の値を使って各都市を地図上の適切な位置に配置しました。苦労した点は、地図のサイズと比率に合わせて、都市の座標を手動で調整しなければならなかったことです。参考文献や講義資料をもとに試行錯誤しながら、都市の緯度経度から対応する地図上の位置を算出し、地道にパーセンテージを調整して正確に表示できるように工夫しました。これにより、見た目にもわかりやすく、誰にでもわかりやすいような天気情報表示ページを作成することができました。また、数当てゲームでは、コードを書くのに少し苦労しました。if文を書くことも苦心しましたが、例えば1回か2回で数字を当ててしまった後に、まだ答えようとすると数字が飛んで4回目となってしまっていました。そこで、isGameOverという変数を作り、ゲームが終わったかどうか判断させてその後ユーザーが入力しても回数が飛ばないように工夫しました。このように自分なりに考え工夫し、この授業に取り組めたと思います。HTML、CSS、JavaScript、DOMは他の講義でもよく耳にしますが、実際に手を動かして体験することで、それぞれの役割や使い方をより深く理解することができました。この内容を学習したことで、プログラミングの勉強がより興味深く、楽しいものになったと感じます。特に自分でWebページを完成させたときの達成感はとても大きく、今後も積極的に取り組んでいきたいと思いました。この経験を活かし、さらに難しい内容の講義や、将来社会に出たときに実践的に活用できるように、この授業の内容を継続して深く勉強していきたいと思います。自分自身の成長にもつながる大変貴重な授業でした。
